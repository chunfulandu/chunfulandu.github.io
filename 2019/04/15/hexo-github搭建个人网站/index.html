hexo+github搭建个人网站

发表于2019-04-15|更新于2021-08-10|工具
|阅读量:

注：下列均在macos环境下安装。

安装git

安装方法简单得一批，不懂的自行了解。

安装node.js

  1. 安装homebrew

    可以通过brew -v来查看是否安装了homebrew，如果能正确显示homebrew的版本号，说明homebrew已经安装。
    如果没有安装，通过下面的命令进行安装：

     ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

  2. 安装node.js，在终端中输入下列命令：

     brew link node
 brew uninstall node
 brew install node

  3. 验证node是否安装成功，npm: node package manager，node.js的包管理工具

     npm -v
 node -v
 # 通过homebrew安装的，执行下面命令，可查看到“node”
 brew -v

  4. 安装cnpm

    npn由于源服务器在国外下载node包经常超时，cnpm使用国内镜像，通过下面命令安装cnpm，然后通过cnpm -v查看是否安装成功，之后便可使用cnpm代替npm了

     npm install -g cnpm --registry=https://registry.npm.taobao.org
 cnpm -v

安装Hexo

  1. 安装hexo

    • 我在使用npm install -g hexo命令进行安装时，出现如下错误：

        fetchMetadata: sill fetchPackageMetaData error for warehouse@^2.2.0 Unexpected end of JSON input while parsing

    • 然后再次进行安装时，又出现下面错误：

        npm ERR! Unexpected end of JSON input while parsing near '...p-ansi-3.0.1.tgz"},"e'

  npm ERR! A complete log of this run can be found in:
  npm ERR!     /Users/chengchf/.npm/_logs/2019-04-14T16_51_51_254Z-debug.log

    • 经google，推荐使用npm cache clean --force命令，但依旧无效，错误如下：

        rollbackFailedOptional: verb npm-session b119e4412863cfbc

    • 再次google，找到解决方法，原因为设置的代理错误，使用下列删除命令，然后使用cnpm的命令进行安装，成功

        npm config rm proxy
  npm config rm https-proxy
  cnpm install -g hexo

  2. 创建存储博客的文件夹，我的存储博客的文件夹命名为chunfulandu_blog，然后进入到该文件夹

     mkdir chunfulandu_blog
 cd chunfulandu_blog/

  3. 初始化本地博客，下载一系列文件，这个过程可能稍微有点慢

     hexo init

  4. 安装依赖包

     cnpm install

  5. 至此本地博客就搭建好了，执行下列命令生成本地网页文件并开启服务器，然后通过http://localhost:4000查看本地博客

     # 同hexo generate
 hexo g
 # 同hexo server
 hexo s

本地博客关联github

  1. 在自己的github上，新建仓库，名称必须为username.github.io，如：chunfulandu.github.io，仓库README、License之类的留空即可，之后在发布上传时会被强制覆盖抹掉

  2. 在本地文件夹下，打开_config.yml文件，修改配置如下，repository为这种形式的是配置了SSH Key之后的，如果没有配置则使用https形式的地址。

     deploy:
   type: git
   repository: git@github.com:chunfulandu/chunfulandu.github.io.git
   branch: master

  3. 为了能够使hexo部署到github上，需安装一个插件

     cnpm install hexo-deployer-git --save

  4. 执行下列命令，然后就可在浏览器通过username.github.io访问你的博客了

     hexo clean
 hexo g
 hexo d
